# Telegram AI/ML Content Aggregator Bot

Бот, который мониторит публичные Telegram-каналы про AI/ML/Data Science, фильтрует посты по ключевым словам, оценивает их релевантность через Qwen (OpenRouter, бесплатно), генерирует краткое саммари и публикует в ваш канал со ссылкой на оригинал.

## Как работает

```
Каналы-источники → Telethon (чтение) → Фильтр ключевых слов → Qwen via OpenRouter (оценка + саммари) → Bot API (публикация)
```

1. Каждые N минут бот проверяет список каналов-источников
2. Новые посты проходят фильтр по ключевым словам (AI, ML, LLM и др.)
3. Прошедшие фильтр посты отправляются в Qwen (через OpenRouter) для оценки релевантности (1-10) и генерации саммари
4. Посты с оценкой >= порога публикуются в целевой канал
5. Все обработанные посты сохраняются в SQLite для дедупликации

## Пререквизиты

### 1. Python 3.11+

```bash
python3 --version
```

### 2. Telegram API credentials (Telethon — чтение каналов)

Нужны для чтения постов из публичных каналов через Telegram Client API.

1. Перейти на https://my.telegram.org
2. Войти по номеру телефона
3. Перейти в **API development tools**
4. Создать приложение (название и описание — любые)
5. Скопировать **api_id** (число) и **api_hash** (строка)

> При первом запуске бота Telethon попросит ввести код подтверждения из Telegram. Сессия сохраняется в файл `session.session` — повторная авторизация не потребуется.

### 3. Telegram Bot Token (Bot API — публикация в канал)

Нужен для отправки саммари в ваш канал.

1. Открыть [@BotFather](https://t.me/BotFather) в Telegram
2. Отправить `/newbot`
3. Задать имя и username бота
4. Скопировать **токен** (формат: `123456789:ABCdefGhIJKlmnOPQrsTUVwxyz`)

### 4. Целевой Telegram-канал

Канал, куда бот будет публиковать саммари.

1. Создать канал в Telegram (или использовать существующий)
2. Добавить бота (из шага 3) в канал как **администратора** с правом **отправки сообщений**
3. Записать username канала (формат: `@my_channel`)

### 5. OpenRouter API Key (бесплатно)

Нужен для оценки релевантности и генерации саммари через модель Qwen 3 32B.

1. Зарегистрироваться на https://openrouter.ai
2. Перейти в **Keys** (https://openrouter.ai/keys)
3. Создать новый ключ
4. Скопировать ключ (формат: `sk-or-v1-...`)

> Бот использует бесплатную модель `qwen/qwen3-32b:free`. Оплата не требуется.

## Установка

```bash
# Клонировать / перейти в проект
cd time-optimize

# Создать виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Скопировать и заполнить переменные окружения
cp .env.example .env
```

## Настройка .env

Открыть `.env` и заполнить все значения:

```env
# Telegram API (шаг 2)
TELEGRAM_API_ID=12345678
TELEGRAM_API_HASH=abcdef1234567890abcdef1234567890
TELEGRAM_PHONE=+79001234567

# Telegram Bot (шаг 3)
TELEGRAM_BOT_TOKEN=123456789:ABCdefGhIJKlmnOPQrsTUVwxyz

# Целевой канал (шаг 4)
TARGET_CHANNEL=@my_ai_digest

# OpenRouter API (шаг 5) — бесплатно
OPENROUTER_API_KEY=sk-or-v1-...

# Интервал проверки (минуты)
CHECK_INTERVAL_MINUTES=5

# Порог релевантности (1-10)
RELEVANCE_THRESHOLD=7
```

## Настройка каналов-источников

Отредактировать список `SOURCE_CHANNELS` в `config.py`:

```python
SOURCE_CHANNELS = [
    "ai_newz",
    "gonzo_ML",
    "data_secrets",
    # добавить свои каналы...
]
```

Указывать username канала без `@`.

## Запуск

```bash
source .venv/bin/activate
python bot.py
```

При первом запуске:
1. Telethon запросит **код подтверждения** из Telegram (придёт в приложение)
2. При наличии двухфакторной аутентификации — запросит **пароль**
3. После авторизации создастся файл `session.session` — больше авторизация не нужна
